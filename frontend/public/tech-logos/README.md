# Логотипи технологій

Сюди кладіть файли логотипів технологій (взуттєвих), на які маєте право.
Каталог автоматично підхопить їх у картці товару в блоці «Технології».

**Формат імені файлу:** `<slug>.svg` (перевага) або `<slug>.png`.
Слаг для кожної відомої технології заданий у `frontend/src/techLogos.ts`.

Поточні слаги (частина): `gore-tex`, `vibram`, `megagrip`, `boost`, `nike-air`,
`zoom-air`, `react`, `gel`, `ortholite`, `primaloft`, `thinsulate`, `contagrip`,
`dri-fit`, `boa`, `flyknit`, `cloudfoam`, `croslite`, `repreve`, `abzorb`,
`meta-rocker`, `simpatex`, `respira`, `spherica`, `cordura`, `ultracush`.

Поки файлу немає — показується охайний текстовий бейдж (нічого не ламається).

**Рекомендації до зображень:**
- Монохром/прозорий фон (щоб гарно лягало на світлу й темну тему).
- Висота лого ~16–18px; SVG масштабуватиметься автоматично.
